# RACE_RunAndCatchEnding
카메라로 당신의 동작을 인식해 캐릭터를 달리게 하는 경주 게임입니다.
  
### 소개
- 1 대 1의 레이싱 게임으로 먼저 결승선에 도달하는 사람이 이깁니다.
- 화면 앙 옆에 여러분의 캐릭터가 달립니다.
- 화면 가운데에서 제시하는 동작을 취해 캐릭터의 속도를 높이세요.
- 일정 시간 내에 동작을 정확히 취하면 콤보가 쌓이며 캐릭터가 더 빨라집니다.
- 상대보다 먼저 결승선에 도착해 우승하세요!
